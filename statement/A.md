## Problem A. A+B


- Time Limit: 2 sec
- Memory Limit: 512 MB


### Problem Statement


Calculate A+B.


### Input


The input consists of a single test case in the format below.


> $A$ $B$


Integers A and B ($1 \le A, B \le 10^9$) are given in one line.


### Output


Print A+B in one line.
